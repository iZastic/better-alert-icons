# Better Alert Icons

Customize the size and position of alert icons on buildings in Factorio.

### Features

* Scale: Change the size of alert icons on buildings
* Positioning: Choose the location of the alert icon for buildings

### Settings

Adjust settings in `Settings > Mod settings > Startup`

* Scale (0.0 - 1.0): size of the alert icon
* Padding (0.0 - 0.3): percentage of building size to use as padding from the edge
* Horizontal Position (left, center, right)
* Vertical Position (top, center, bottom)
